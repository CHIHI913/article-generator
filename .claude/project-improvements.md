# 改善履歴・教訓

## 実装済み改善

### 2025-06-15: ストリーミング実装の修正
**問題**: 初期実装でストリーミングが正常に動作しなかった  
**原因**: `useCompletion`の`streamProtocol`設定と APIルートの`toTextStreamResponse()`の不一致  
**解決策**: 
- フロントエンド: `streamProtocol: 'text'`
- バックエンド: `result.toTextStreamResponse()`
**教訓**: Vercel AI SDKのストリーミング設定は厳密な対応関係が必要

### 2025-06-15: プロンプト・リクエストボディキーの統一
**問題**: フロントエンドとバックエンドでリクエストキーが不一致（`prompt` vs `overview`）  
**原因**: 開発初期のキー名変更時の修正漏れ  
**解決策**: バックエンドで両方のキーに対応する後方互換性を実装
```typescript
const ov = overview ?? prompt ?? '';
```
**教訓**: API設計時のキー名は最初から統一し、変更時は影響範囲を慎重に確認

### 2025-06-15: バリデーション強化
**問題**: 空の概要での生成リクエスト  
**解決策**: 
- フロントエンド: `minLength={10}` + `required`属性
- バックエンド: 10文字未満のリクエストを400エラーで返却
**教訓**: クライアント・サーバー両方でのバリデーションが必要
