# OpenHands リポジトリマイクロエージェント定義

## リポジトリ概要
本リポジトリは Next.js (App Router) と Vercel AI SDK を用いた **記事生成アプリ** です。ユーザーが記事概要を入力し、フォーマットとモデルを選択すると、AI が指定フォーマットに沿った記事ドラフトを生成します。

## ローカル環境での実行方法
```bash
# 依存関係インストール
npm ci

# 環境変数を用意
cp .env.example .env.local
# .env.local に OpenAI などの API キーを設定

# 開発サーバー起動
npm run dev
# → http://localhost:3000 で確認
```
